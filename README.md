# Aaditya Golash Business Technology Portfolio

Static, Vercel-ready portfolio site for Aaditya Golash, focused on business technology strategy, analytics, product operations, automation, and implementation work.

## Local Development

```bash
npm run dev
```

Open `http://localhost:5173`.

## Production Build

```bash
npm run build
npm run preview
```

The production output is generated in `dist/`.

## Vercel

- Framework Preset: `Other`
- Build Command: `npm run build`
- Output Directory: `dist`

The build script copies `index.html`, `styles.css`, and `assets/` into `dist/`.
