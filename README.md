# Aaditya Golash Business Technology Portfolio

Static portfolio site for Aaditya Golash, focused on business technology, analytics, product operations, and implementation work.

## Local Development

Open `index.html` directly in a browser, or run the build command to create the Vercel output folder:

```bash
npm run build
```

## Vercel

- Build Command: `npm run build`
- Output Directory: `dist`

The build script copies `index.html`, `styles.css`, and `assets/` into `dist/`.
